# Food Finder Application

Application to help find food near the office. Users can get random restaurants, search for a specific type of restaurants and view specific information a restaurant.

## Technical Stack

- ReactJS / Typescript
- ViteJS
- ChakraUI
- ExpressJS
- NodeJS
